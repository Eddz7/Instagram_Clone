## Features

- Responsive UI
- Email and password authentication
- Post creation/deletion
- Follow/unfollow functionality
- Display user posts
- Comment functionality
- Follow/following count
- Sign out
- Updates in real time

## Technologies Used

### Server Side
- Firebase Auth
- Firebase Storage
- Firebase Firestore

### Client Side
- Flutter
- Provider
